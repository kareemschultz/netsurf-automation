# Configure WhatsApp Business API integration

## Description
Set up WhatsApp Business API for customer messaging through Chatwoot.

## Prerequisites
- Facebook Business account
- WhatsApp Business API access
- Verified phone number

## Steps
1. Create WhatsApp Business App in Meta Developer Console
2. Configure webhook URL: `https://chat.ntpowergy.com/webhooks/whatsapp`
3. Add WhatsApp inbox in Chatwoot
4. Update n8n workflows with WhatsApp credentials

## Acceptance Criteria
- [ ] WhatsApp Business API configured
- [ ] Chatwoot inbox connected
- [ ] Test messages sending/receiving
- [ ] n8n lead capture workflow active
