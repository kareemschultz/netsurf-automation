# Set up Directus collections for business content

## Description
Create Directus collections to manage business content for AI responses and website.

## Collections to Create

### Products
- name, sku, category, description, price, specifications, images

### FAQ
- question, answer, category, business (power/park)

### Canned Responses
- trigger_keywords, response_template, category

### Business Hours
- business, day, open_time, close_time, is_holiday

### Promotions
- title, description, start_date, end_date, discount_percent

- [ ] All collections created with proper fields
- [ ] Sample data imported
- [ ] API permissions configured
- [ ] n8n integration tested
