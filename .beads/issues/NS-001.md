# Configure n8n PostgreSQL credentials

## Description
Create PostgreSQL credential in n8n to enable workflow database operations.

## Steps
1. Open n8n at https://n8n.ntpowergy.com
2. Go to Credentials â†’ Add Credential
3. Select PostgreSQL
4. Configure:
   - Name: `Netsurf PostgreSQL`
   - Host: `postgres`
   - Database: `netsurf`
   - User: `netsurf`
   - Password: (from POSTGRES_PASSWORD in .env)
   - Port: `5432`
5. Save and test connection

## Acceptance Criteria
- [ ] PostgreSQL credential created in n8n
- [ ] All workflows updated to use the credential
- [ ] Test connection successful
